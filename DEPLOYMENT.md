# Deployment

## Render Backend

Use the root `render.yaml` blueprint.

Required production env vars:

```env
APP_ENV=production
APP_DEBUG=false
APP_KEY=<generated by Render>
DB_CONNECTION=pgsql
DB_URL=<Render Postgres connection string>
FRONTEND_URL=https://<your-vercel-app>.vercel.app
```

Optional agent env vars:

```env
HERMES_WEBHOOK_URL=
OPENCLAW_WEBHOOK_URL=
SLACK_WEBHOOK_URL=
SLACK_AGENT_CHANNELS=sprint-main,agent-coder,agent-log
```

The Render start command runs migrations and starts Laravel on `$PORT`.

## Vercel Frontend

Set project root:

```text
kanban-frontend
```

Set env:

```env
VITE_API_URL=https://<your-render-api>.onrender.com/api
```

Build command:

```bash
npm run build
```

Output directory:

```text
dist
```
